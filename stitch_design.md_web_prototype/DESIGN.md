# 二鱼的猫窝 · 网站设计文档

> 用于 Stitch 原型设计的完整规格说明

**文档版本**：v1.0  
**最后更新**：2026-04-26

---

## 一、项目概述

### 1.1 项目名称

**二鱼的猫窝**（域名：eryudemaowo.com）

### 1.2 一句话定位

一位写作者用文字喂养小猫的地方——以作品承载为核心、以养猫式打卡为乐趣、逐步沉淀成个人品牌的轻量级写作者个人站。

### 1.3 核心用户

| 类型 | 说明 |
|------|------|
| **主要用户（站主）** | 写作者本人，需要承载作品、记录阅读、自我激励。 |
| **次要用户（访客）** | 读者、写作爱好者、潜在编辑/品牌方。 |

### 1.4 核心价值主张

- 让作品有「家」，不再因投稿失败而消失。
- 把孤独的写作过程游戏化（养猫机制）。
- 沉淀个人品牌资产，对外展示完整的创作者形象。

---

## 二、视觉设计规范

### 2.1 整体风格

**文艺 + 小俏皮**

- **文艺感**：米白底色、衬线字体、留白、手写感细节。
- **俏皮感**：圆润的卡片、卡通猫元素、俏皮的导航命名、轻微动效。

### 2.2 配色方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| 主背景色 | 米白 / 暖奶白 | `#FBF7F0` |
| 卡片底色 | 纯白 | `#FFFFFF` |
| 主文字色 | 深棕灰 | `#3D3530` |
| 次要文字色 | 暖灰 | `#8B7E74` |
| 主题强调色（橘猫橙） | 暖橘 | `#E89B5C` |
| 辅助色 1（奶茶棕） | 浅咖 | `#D4B896` |
| 辅助色 2（薄荷绿） | 嫩绿 | `#A8C8A6` |
| 高亮 / 进度色 | 蜜糖黄 | `#F4C95D` |
| 边框 / 分隔线 | 浅米 | `#EFE6D8` |

### 2.3 字体规范

| 用途 | 字体 |
|------|------|
| 中文标题 | 思源宋体 / Noto Serif SC（粗） |
| 中文正文 | 思源宋体 / Noto Serif SC（常规） |
| 英文标题 | Playfair Display 或 Georgia |
| 英文正文 | Georgia |
| 装饰 / 手写感 | 霞鹜文楷（用于副标题、引言） |

**字号建议**：标题 28–36px、副标题 18–22px、正文 16px、辅助信息 13–14px。

### 2.4 视觉元素与质感

- **圆角**：卡片统一使用 16–24px 大圆角，营造柔软感。
- **阴影**：浅色柔和阴影，避免硬边。
- **插画风格**：扁平卡通猫咪（站主后期会自行设计一套表情包，目前用占位）。
- **图标风格**：lucide-react 线性图标，细线条。
- **装饰元素**：手绘感的小鱼、猫爪印、虚线分隔、星星点缀。

### 2.5 动效原则

- 节制、舒适，不抢戏。
- 关键交互必有反馈（如喂猫粮的飘浮动画）。
- 页面切换柔和淡入。

## 四、页面详细设计

### 4.1 首页 `/`

**目标**：访客 5 秒内理解这是谁、在做什么、有什么内容。

| 模块 | 说明 |
|------|------|
| **1. Hero 区（首屏）** | 左：站主头像（圆形，可用猫咪插画代替）；右：一句话自我介绍 + 笔名「二鱼」；装饰：背景有淡淡的鱼骨/猫爪线稿；底部小字：当前正在养第 N 只猫 · 累计写作 X 字。 |
| **2. 当前养猫进度卡** | 居中大卡片；当前猫咪 SVG/插画（根据成长阶段显示不同形态）；猫名（待养成时显示「未命名」）；进度条 + 字数（如 32,500 / 50,000 字粮）；「去字房写字」按钮 → 跳转作品集。 |
| **3. 本月写作目标卡** | 横向卡片；进度条（薄荷绿色）；文案示例：本月目标 10,000 字 · 已完成 6,200 字 · 还差 3,800 字。 |
| **4. 最新作品（3 篇）** | 横向卡片；每张：分类标签（小说/散文/评论）、标题、首段摘要、字数、日期。 |
| **5. 最近读完（3 本）** | 横向卡片；每张：书封面、书名、作者、我的评分（5 颗鱼骨头）、一句短评。 |
| **6. 页脚** | 友链 · 邮箱 · RSS · 版权声明。 |

### 4.2 字房（作品集）`/works`

**目标**：清晰展示所有作品，按类型筛选。

1. **页面标题区**：「字房 · 文字栖居之所」；副标题：本站累计 X 篇作品 · X 万字。
2. **分类筛选 Tab**：全部 / 小说 / 散文 / 评论；当前选中 tab 显示橘色下划线。
3. **作品列表**：卡片网格（桌面 2 列，移动 1 列）。每张卡片：顶部分类标签；标题；首段摘要（2 行省略）；底部：字数 · 阅读时长 · 创作日期。
4. **空状态**：小猫插画 +「这里还空空的，正在酝酿……」

### 4.3 单篇阅读页 `/works/[slug]`

**目标**：提供舒适的长文阅读体验。

1. **文章头部**：分类标签；大标题（居中）；副标题/题记（斜体）；元信息行：日期 · 字数 · 阅读时长。
2. **正文区**：最大宽度约 680px，居中；衬线字体、行高 1.8、段落间距充足；支持图片、引用块、分隔符。
3. **作者手记（可选）**：与正文有视觉分隔（虚线 + 小猫图标）；浅米色背景卡片；标题「🐱 写在故事之外」；创作背景、灵感来源等。
4. **文末互动区**：「如果喜欢，给二鱼一颗小鱼干 🐟」按钮（点赞）；字数与喜爱数显示。
5. **上一篇 / 下一篇** 导航。
6. **评论区（Giscus 嵌入）**：标题「📮 留点什么吧」。

### 4.4 书架 `/bookshelf`

**目标**：仿微信读书的视觉，承载读书记录。

1. **顶部统计条**：2026 年已读 X 本 · 累计 XX 万字 · 平均评分 X.X。
2. **Tab 切换**：在读 · 已读 · 想读；数字显示在 tab 名后方（如「已读 23」）。
3. **书籍网格**：桌面 4 列、平板 3 列、移动 2 列。每本书卡片：书封面（带阴影、轻微倾斜拟物）；书名；作者；评分（鱼骨头 1–5）；「在读」时显示进度条；悬停：卡片轻微上移 + 阴影加深。
4. **笔记页** `/bookshelf/[slug]`：大书封 + 元信息 + 完整短评；长篇书评/读书笔记（Markdown）；摘抄列表（关联金句墙）。

### 4.5 字粮 · 养猫系统 `/cats` ⭐ 核心特色

**目标**：把写作字数游戏化，提供长期激励。

**核心机制**

- 每写 1 字 = 1 颗猫粮；50,000 字养成 1 只猫；养成的猫永久陈列在猫窝；自动从作品 frontmatter 字数累加。

**模块**

1. **页面标题区**：「字粮 · 猫咪养成日记」；引言：「每一个字，都是喂给小猫的一颗粮」。
2. **当前养猫主区（巨大卡片）**：渐变背景（米色到淡橘）；中央：当前猫咪插画（根据阶段大小不同）；阶段标签：奶猫 / 幼猫 / 少年猫 / 即将成年 / 成年；可爱口吻状态文案；进度条 + 字数；「为这只猫起名」输入框（养成时激活）。
3. **成长阶段说明（横向时间线）**：五节点 0 / 5,000 / 20,000 / 40,000 / 50,000；每节点小图标 + 阶段名；当前阶段高亮。
4. **猫窝 · 已养成的猫**：网格；每只猫卡片：插画、名字、「陪我写完了《XX》」、养成日期、累计陪伴字数、撸猫按钮（点击动画 + 计数）、悬停猫咪轻微晃动。
5. **统计条（页底）**：累计养成 X 只猫、累计字数 XX 万、写作天数 X 天、最长连续 X 天。

### 4.6 金句墙 `/quotes`

**目标**：Pinterest 式金句卡片墙。

1. **页面标题区**：「金句墙 · 偷来的星光」；累计 X 张卡片。
2. **标签筛选条（横向滚动）**：全部 · 生死 · 爱情 · 孤独 · 自然 · 写作 · 时间 …；选中高亮。
3. **瀑布流卡片网格**：桌面 3 列、平板 2 列、移动 1 列。每张卡片：顶部装饰引号；金句正文（衬线）；虚线分隔；出处：《书名》— 作者；我的感想（小字、斜体，可选）；标签胶囊；右下角导出图片（小相机）；卡片底色随机轻微变化（米白、淡黄、淡绿、淡粉）。
4. **点击卡片**：大图模态框，适合分享截图；底部「另存为图片」。

### 4.7 留言板 `/guestbook`

1. 小猫插画；「欢迎在猫窝门口留个爪印 🐾」；引导文案：读后感、问候、推荐书等。
2. Giscus 评论区，与站点风格一致。

### 4.8 关于 `/about`

1. 个人介绍：头像 + 笔名「二鱼」；自由风格自我介绍。
2. 写作理念：引言式短句。
3. 作品成绩（可空）：已发表、获奖、媒体报道。
4. 联系方式：邮箱、社交媒体（可选）。
5. 关于本站：技术栈致谢、建站日志（创建于 X 年 X 月）。
6. 友链：其他写作者博客。

## 五、组件设计规范

### 5.1 通用卡片组件

- 圆角 20px、白底、浅阴影；内边距 24px。
- 悬停：轻微上移 4px + 阴影加深。

### 5.2 按钮

- **主按钮**：橘色 `#E89B5C` 底 + 白字 + 圆角 12px。
- **次按钮**：白底 + 棕字 + 棕色边框。
- **文字按钮**：仅文字 + 下划线悬停。

### 5.3 标签胶囊

- 圆角 999px（药丸形）；浅色底（米色/淡橘/淡绿）+ 深色字；字号 12–13px。

### 5.4 进度条

- 高度 12px，圆角 999px；底色 `#EFE6D8`；进度色 `#E89B5C` 或 `#F4C95D`；进度文字在右侧或上方。

### 5.5 评分图标

- 用小鱼骨头 / 小鱼代替星星；5 个一组，已选橘色，未选浅灰。

## 八、Stitch 优先级

1. **首页** — 整体风格定调，视觉锚点  
2. **字粮养猫页** — 网站灵魂、最具特色  
3. **单篇阅读页** — 阅读体验与停留时间  
4. **金句墙** — 视觉密度高、风格体现  
5. **作品集** — 列表页样板  
6. **书架** — 与金句墙类似的网格形态  
7. **关于 / 留言板** — 简单页面，最后做  

---

## 九、设计禁忌

- ❌ 不要过度性冷淡的纯黑白灰  
- ❌ 不要过艳过饱和（保持柔和）  
- ❌ 不要过多动画干扰阅读  
- ❌ 正文不要用无衬线（破坏文艺感）  
- ❌ 不要用尖锐直角卡片（破坏俏皮感）  
- ❌ 猫咪不要写实到恐怖谷
